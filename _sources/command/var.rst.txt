.var
----

``.var`` displays variables.

Example
~~~~~~~

.. code-block:: text

	-- 0/2 - 0 ----------------------------------------------------------------
	.var
	activityCount                 The number of rows in the query result
	display                       Display object
	esc                           Escape object
	globals                       Global objects
	interpreter                   Interpreter object
	parent                        Parent variables
	session                       Current session object
	-- 0/3 - 0 ----------------------------------------------------------------
